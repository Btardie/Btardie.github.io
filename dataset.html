Welcome to the Dataset Generator Tool!

This tool allows you to create custom datasets tailored to your specific needs. Here's how to use it:

1. Specify the Number of Records: Start by entering the total number of records you want in your dataset. This determines the size of the dataset to be generated.

2. Add Columns with Desired Data Types: Use the "Add Column" button to add each column you need in your dataset. You can specify various data types:

  • Text: Create columns with randomized text values. You can enter sample text values, and the tool will randomly select one for each record.
  • Number: Generate columns with random numbers. You can set the minimum and maximum range for numeric values.
  • Date: Add date columns with randomized dates. Specify a start and end date to create a range, and the tool will generate dates within this interval.
  • Category: Create categorical columns with predefined options. Enter category values, and the tool will randomly select one for each row.

3. Interactive Preview: As you configure your columns, the preview table on the right will update automatically, showing the first 15 rows of your dataset. This interactive preview helps you visualize the structure and content of your dataset before downloading.

4. Download Full Dataset: Once satisfied with your dataset setup, click "Download Dataset" to download a CSV file containing the full dataset based on the specified number of records.

This tool is perfect for creating sample data for testing, prototyping data analysis workflows, or generating realistic datasets for training machine learning models.
