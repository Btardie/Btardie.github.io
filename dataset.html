Welcome to the Dataset Generator Tool!

Step 1: Specify the Number of Records

Enter the total number of records you want in your dataset. This defines the size of the dataset.

Step 2: Add Columns and Choose Data Types

Click "Add Column" to create a new column. You can specify different data types for each column:

  • Text: Enter sample text values separated by commas, and the tool will randomly select one for each row.
  • Number: Define a range (minimum and maximum values), and the tool will generate random numbers within this range.
  • Date: Specify a start and end date, and the tool will generate random dates within this range.
  • Category: Provide categories (e.g., "Red, Blue, Green"), and the tool will randomly assign one to each row.

Step 3: Interactive Preview

The preview table on the right updates automatically, showing the first 15 rows of your dataset as you make changes. This preview helps you check your setup before downloading.

Step 4: Download Full Dataset

Once you're satisfied with your setup, click "Download Dataset" to download the complete dataset as a CSV file. The file will contain all records based on your specified number of rows.

This tool is perfect for generating sample data for testing, prototyping data analysis, or creating realistic datasets for machine learning models.
