Welcome to the Dataset Generator Tool!

This tool allows you to create custom datasets tailored to your specific needs. Whether you're working on data science projects, testing machine learning models, or building prototypes, this generator can provide you with sample data in a few simple steps.

How to Use This Tool

  1. Specify the Number of Records: Enter the total number of rows you want in your dataset. This determines the size of the dataset.
  2. Add Columns with Desired Data Types: Click the Add Column button to create new columns. For each column, select the data type and enter relevant details.
    • Text: Provide sample text values separated by commas (e.g., Apple, Orange, Banana) and optionally assign probabilities for each value to control their frequency.
    • Integer / Float: Define a minimum and maximum range (Basic) or specify a mean and standard deviation (Advanced) to generate random numbers within the range.
    • Date: Set a start and end date to create a range. The tool will generate dates within this interval.
    • Category: Enter categorical values (e.g., High, Medium, Low) with optional probabilities to assign weights for each category.
  3. Preview Your Dataset: As you configure your columns, the preview table will automatically update on the right, showing the first 15 rows of your dataset. This preview helps you check your setup before downloading.
  4. Download Your Dataset: Once you're satisfied with your dataset setup, click Download Dataset to get a CSV file containing your generated data.

What You Can Use This For

This dataset generator is ideal for a variety of applications, including:

  • Machine Learning and Data Science: Generate sample data to train, test, and evaluate machine learning models without using real-world data. This is especially useful when working with classification, regression, and clustering algorithms.
  • Database and Software Testing: Create sample data to populate databases and test software applications. This can help developers ensure that their systems handle data correctly.
  • Prototyping and Demo Data: Quickly produce data to use in prototypes and demonstrations. This can help stakeholders visualize how data-driven applications will function with realistic datasets.
  • Exploratory Data Analysis (EDA): Practice data analysis techniques, statistical calculations, and visualizations using controlled datasets before working with real-world data.
